# Winnsen Canvas Data

Runtime canvas data is stored in the browser localStorage key `winnsen-product-canvas-v1`.

To move work between computers:

1. Open the app on the source computer.
2. Click `导出` in the top bar to download `winnsen-product-canvas.json`.
3. Copy the JSON file to the target computer or commit it into this `data/` folder if it is safe for the company.
4. Open the app on the target computer and click `导入`.

Do not commit confidential customer, pricing, structure, or unreleased product data unless the repository access is approved for that data.
